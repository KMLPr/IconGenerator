Name: Icon Generator

A JavaFX application that generates simple, random geometric icons with different shapes, colors, and symmetry options.  
Perfect for placeholders, avatars, or just for fun!

- Random geometric shapes: circle, square, triangle
- Symmetry options: vertical, horizontal, radial
- Random color palettes
- Export icons as PNG (planned)

- Java SDK 24
- JavaFX 23
- Maven
- IntelliJ IDEA

Work in progress for the time being
